# Jenkins Docker

```bash
# edit files
# vim terraform/terraform.tfvars
# vim ansible/inventory
# vim Makefile (change image tag)

# create a spot instance on AWS
make spot

# create a jenkins
make

# push jenkins image
docker push "${image}"

# deploy jenkins on AWS
make deploy
```
